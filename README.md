Smart Captioner is a web-based application that helps users generate accurate, multilingual subtitles for videos using AI.

## Features
- Upload video files (MP4, MKV, AVI, MOV)
- Select subtitle language (English, Malayalam, Hindi, Tamil, Telugu)
- AI-powered transcription using Whisper
- Subtitle preview
- Download SRT files
- Download final video with embedded subtitles

## Tech Stack
- Python
- Flask
- OpenAI Whisper
- FFmpeg
- HTML
